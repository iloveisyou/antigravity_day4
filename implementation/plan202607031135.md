# Hanes.md Step 5 구현 계획 (LocalStorage 연동)

일기 내용과 AI 분석 답변을 브라우저의 로컬 스토리지(LocalStorage)에 영구 저장하여, 사용자가 새로고침을 하거나 웹페이지를 다시 방문했을 때 마지막 작성 내용 및 분석 결과를 유지하도록 구현합니다.

## 구현 상세

### 1. 로컬 스토리지 데이터 로드
- `DOMContentLoaded` 시점에 `localStorage.getItem('diary_text')`와 `localStorage.getItem('diary_ai_response')`를 확인합니다.
- 데이터가 있을 경우 각각 `diaryInput.value` 설정, 글자 수 카운터 업데이트 및 `aiResponseBox` 렌더링 작업을 수행합니다.

### 2. 로컬 스토리지 데이터 저장
- `analyzeBtn` 클릭 후 Gemini API의 결과 수신이 성공한 시점에 `localStorage.setItem`을 이용해 저장합니다.
- 입력창 내용(`text`)과 AI 응답 텍스트(`result.text`)를 저장합니다.
